# rust-dmg
### A gameboy emulator written in rust <br/>
working but currently still in development
### Fetures <br/>
- passes all braggs cpu tests <br/>
- MBC0 and MBC1
- working timers
- working gpu
- plays tetris

### Usage <br />
```cargo run``` *rom path*
